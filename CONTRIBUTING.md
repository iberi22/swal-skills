# Contributing

## Alcance
Este proyecto acepta contribuciones que:
- Sigan la arquitectura definida en AGENTS.md
- Incluyan tests
- Mantengan compatibilidad hacia atrás

## Proceso
1. Crea un issue describiendo el cambio
2. Fork y branch: `fix/descripcion` o `feat/descripcion`
3. Implementa con tests
4. PR contra `main` con descripción clara

## Estándares
- Código limpio y comentado
- Sin secrets hardcodeados
- Una feature por PR
